# 🌤 10-Day Weather Forecast UI Component

A professional and lightweight weather forecast interface designed to demonstrate expert-level **CSS Flexbox** alignment and **UI/UX design** principles.

## 🚀 Key Features:
- **Pure HTML5 & CSS3:** No external frameworks used, ensuring 100% custom and optimized code.
- **Flexbox Architecture:** Modern layout system for perfect alignment without hardcoded margins.
- **Pixel-Perfect UI:** Focused on white space, readable typography (Roboto), and aesthetic color palettes.
- **Mobile Responsive:** Fully optimized for seamless viewing across all devices and screen sizes.

## 🛠 Tech Stack:
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom properties, transitions, and Flexbox layouts.
- **Google Fonts:** Integrated 'Roboto' for a modern, clean look.

## 📁 Project Structure:
- `index.html`: The main structural file.
- `style.css`: Custom styling and responsive logic.
- `/assets`: Contains weather icons and images.
